# Instagram Clone In Flutter

This is the simple Instagram Clone build using Flutter and Firebase.




## Features

- Authentication using Firebase Auth
- Follow and Following
- Post Uploading with description.
- Like, Comment, Delete
- Profile Upload
- Search Other Users
